Assembly

Assemblery é uma linguagem de baixo nível usada principalmente para programar microcontroladores e/ou microprocessadores. Nosso emulador é baseado na arquitetura do Intel x86 16 bits.

Registradores Gerais

Os registradores são usados utilizados nas operações aritméticas e lógicas que ocorrem na Unidade Lógica Artimica, dos registradores que o x86 possui iremos trabalhar com 4 que são de propósito geral, sendo eles:

AX (16 bits) : que são dividos em: AH (8 bits) - Parte alta AL(8 bits) - Parte baixa

BX(16 bits) : que são divididos em: BH(8 bits) BL(8 bits)

CX(16 bits) : que são divididos em: CH(8 bits) CL(8 bits)

DX(16 bits) : que são divididos em: DH(8 bits) DL(8 bits)

REGISTRADORES DE ÍNDICES E APONTADORES

IP (instruction pointer) OU PC é um registrador utilizado para localizar a posição, dentro do segmento de código e armazenando qual sera a próxima que deve ser instrução a ser executada.

REGISTRADORES SINALIZADORES - FLAGS

Indicam o estado do processador durante a execução de cada instrução, analisando o processamento bit a bit.

Carry Flag = CF
Parity Flag = PF
Auxiliary Carry = AF
Zero Flag = ZF
Sign Flag = SF
Overflow Flag = OF

CONJUNTO DE INSTRUÇÕES ISA – (Instruction Set Architecture)

As instruções (mnêmonicos) disponiveis no nosso simulador são:

MOV

- Usada para transferir dados entre destino e fonte para efetuar essa intrução é necessário:

– registrador e registrador
– registrador e imediato

PILHA

Usada para armazenar o conteudo de um registrador em uma pilha.

SP (stack pointer -> apontador de pilha) é utilizado para acessar a área de pilha na memória; aponta para o topo da pilha.

ADD

- Intrução Lógica Aritmética entre destino e fonte para efetuar essa intrução é necessário:
– registrador e registrador
– registrador e imediato

SUB

-Intrução Lógica Aritmética entre destino e fonte para efetuar essa intrução é necessário: – registrador e registrador – registrador e imediato

POP

- Usada para retirar o conteúdo da pilha para um registrador de 16 bits.

PUSH

- Usada para colocar o conteúdo de um registrador de 16 bits na pilha.
